# getAssets{#getassets}

Restituisce le risorse da IPS.

Sintassi

## Tipi di utenti autorizzati {#section-4673c1c9f4314160af8b165eb2dd20cc}

* `IpsUser`
* `IpsAdmin`
* `IpsCompanyAdmin`
* `TrialSiteAdmin`
* `TrialSiteUser`
* `ImagePortalAdmin`
* `ImagePortalUser`
* `ImagePortalContrib`
* `ImagePortalContribUser`

>[!NOTE]
>
>Restituisce solo le risorse a cui l’utente ha accesso.
