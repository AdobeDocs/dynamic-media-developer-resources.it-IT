|---|---|---|
| maskHandle | `xsd:string` | Handle maschera. |
| nome | `xsd:string` | Nome maschera. |
| maskPath | `xsd:string` | Percorso relativo della maschera. |
| maskFile | `xsd:string` | File maschera. |
