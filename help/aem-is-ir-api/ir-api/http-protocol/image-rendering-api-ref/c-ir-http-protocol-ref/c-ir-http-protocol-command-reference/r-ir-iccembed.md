# iccEmbed{#iccembed}

Incorpora profilo ICC. Specifica se incorporare nell&#39;immagine di risposta il profilo colore ICC funzionante o il profilo specificato con icc=.

`iccEmbed=0|1`

## Proprietà {#section-45c08073626a4cc7823d423dc35b4479}

Può verificarsi ovunque all’interno della richiesta.

## Predefinito {#section-68ed1f0c68dc46b58972804799c277b7}

`iccEmbed=0`, per non incorporare profili ICC nelle immagini di output.
