# Comandi di selezione{#selection-commands}

Questi comandi vengono utilizzati per selezionare gruppi di vignettatura, oggetti, sottoaree di gruppi o oggetti.

Il comando o il materiale, o entrambi, vengono specificati dopo questo comando di selezione e prima che il comando di selezione successivo (o la fine della richiesta) venga applicato all&#39;elemento selezionato. I comandi di selezione delimitano i segmenti di specifica materiale (MSS).
