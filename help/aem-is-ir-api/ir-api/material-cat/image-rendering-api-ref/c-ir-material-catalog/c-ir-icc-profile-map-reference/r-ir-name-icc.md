# Nome{#name}

Nome collegamento profilo. Un nome di scelta rapida per il profilo colore.

## Proprietà {#section-52a4b91bbfa045c49c9541444fad2027}

Stringa di testo. Obbligatorio. Chiave di indice primaria per la mappa di profilo ICC. Ogni `icc::Name` il valore deve essere univoco all’interno della tabella e non deve contenere i caratteri &quot;,&quot;.

## Predefinito {#section-9d23d9b082f541b5b29b4ff92e942942}

Nessuno.
