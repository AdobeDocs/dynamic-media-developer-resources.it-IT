# Tipo{#type}

Tipo di materiale. Tipo di superficie del materiale. Utilizzato insieme a catalog::Gloss e catalog::Roughness per controllare gli effetti di rendering della riflessione 3D.

## Proprietà {#section-86e8bc194f764c848e0ee55630a5ac1b}

Enum. Facoltativo per tutti i materiali. Ignorato se la vignettatura non dispone di funzionalità di rendering riflesso 3D.
