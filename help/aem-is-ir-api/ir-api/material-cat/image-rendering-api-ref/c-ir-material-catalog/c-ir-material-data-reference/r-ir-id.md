# ID{#id}

Identificatore record catalogo. Valore chiave in base al quale i record di materiale vengono cercati dal server di rendering. In genere si tratta di un identificatore di materiale breve e univoco, ad esempio un numero SKU.

## Proprietà {#section-9b20bf2545314620808b4210bce7afc9}

Stringa di testo. Obbligatorio. Ogni valore `catalog::Id` deve essere univoco all&#39;interno di questo catalogo dei materiali e non deve contenere i caratteri &#39;,&#39;.

## Predefinito {#section-6a517046c43a40a7ba0af0212b5f31d2}

Nessuno.
