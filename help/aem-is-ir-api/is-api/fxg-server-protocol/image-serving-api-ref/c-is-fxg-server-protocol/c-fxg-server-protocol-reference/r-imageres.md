# imageRes{#imageres}

Imposta la risoluzione dei bitmap incorporati in una richiesta Web-To-Print.

` imageRes=< *`numero`*>`

Si consiglia 300 per la stampa. Maggiore è il numero, maggiore è il tempo necessario per eseguire il rendering dell’immagine.
