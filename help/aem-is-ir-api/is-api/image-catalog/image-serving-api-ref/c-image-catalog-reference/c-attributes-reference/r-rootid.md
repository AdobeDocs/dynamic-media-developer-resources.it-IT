# RootId{#rootid}

Identificatore del catalogo. L’elemento percorso HTTP utilizzato per identificare questo catalogo nello specificatore oggetto di un’immagine di richiesta.

## Proprietà {#section-9a49da71de634378a06d2347790898a0}

Valore stringa di testo. Può includere solo caratteri validi nei percorsi HTTP.

## Predefinito {#section-c5296f4e52394984bf1c0d265ecde940}

Nessuno. Ogni catalogo deve avere un valore `attribute::RootId` univoco. [!DNL default.ini] ha in genere un `attribute::RootId` vuoto.
