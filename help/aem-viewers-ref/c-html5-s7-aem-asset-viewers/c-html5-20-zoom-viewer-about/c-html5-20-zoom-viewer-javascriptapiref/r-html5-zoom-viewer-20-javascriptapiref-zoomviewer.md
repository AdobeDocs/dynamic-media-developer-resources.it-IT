# ZoomViewer{#zoomviewer}

Riferimento API di JavaScript per Visualizzatore zoom.

`ZoomViewer([config])`

Costruttore, crea un&#39;istanza di Visualizzatore zoom.

## Parametri {#section-4fb77a645fdd45b3aaa5079c31e3bb05}
