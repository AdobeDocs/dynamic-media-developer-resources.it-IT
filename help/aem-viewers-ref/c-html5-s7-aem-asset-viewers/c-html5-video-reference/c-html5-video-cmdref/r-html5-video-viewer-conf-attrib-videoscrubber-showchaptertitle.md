# VideoScrubber.showchaptertitle{#videoscrubber-showchaptertitle}

Attributo di configurazione per Visualizzatore video.

`[VideoScrubber.|<containerId>_videoScrubber.]showchaptertitle=0|1`

<table id="table_C616483932C2482CA9794DDD7313FD7C"> 
 <tbody> 
  <tr> 
   <td colname="col1"> <p> <span class="codeph"> 0|1</span> </p> </td> 
   <td colname="col2"> <p> Disattiva o attiva la barra del titolo dell'etichetta del capitolo video. L'ora di inizio del capitolo non viene visualizzata quando la barra del titolo è disattivata. </p> </td> 
  </tr> 
 </tbody>
