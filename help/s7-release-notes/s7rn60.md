# Note sulla versione di Scene7 Publishing System 6.0{#scene-publishing-system-release-notes}

Note sulla versione che descrivono le funzioni, i miglioramenti e le correzioni per Adobe Scene7 Publishing System 6.0, componente di Adobe Marketing Cloud.

## Nuove funzioni e miglioramenti {#section-4bbd330df5a043f7a56eee619ef7c442}

**Generali**

* Supporto linguistico per il cinese.
* I miglioramenti amministrativi includono:

   * Esportare l’elenco degli utenti aziendali.
   * Inviare e-mail in Ccn agli utenti.
   * Reimpostare le password utente su una password casuale che un utente può reimpostare.

* Schemi di metadati

   Gli schemi di metadati definiscono set di proprietà di metadati. Ogni proprietà dello schema Metadati ha un tipo di dati di base come Stringa, Mobile, Intero, Data o Sì/No. È inoltre possibile definire elenchi, scelte aperte o scelte chiuse. Gli schemi di metadati possono essere utilizzati in tutti i flussi di lavoro relativi ai metadati, come ricerca, modifica, importazione, visualizzazioni di metadati e predefiniti per metadati. <!-- [More information](http://help.adobe.com/en_US/scene7/using/WS259993e42159a215-1c6a66df1265272619e-7fec.html#WSd968ca97bf03cf72-5e3dd3a113268dc80f5-8000). -->

   Gli schemi di metadati sono la tecnologia che sostituisce le UDF e sono destinati a flussi di lavoro di metadati personalizzati, in particolare nel contesto del server di metadati.

* Metadata Server

   >[!NOTE]
   >
   >Il server metadati sarà disponibile a partire da maggio. Deve essere attivato dal supporto. Il server metadati è attualmente disponibile solo in Nord America ed Europa.

* Il server metadati espone un sottoinsieme di tutti i campi di metadati di una risorsa a un server rivolto al pubblico e può essere richiesto qui da pagine o server eCom. I metadati esposti possono essere un insieme combinato di proprietà di Asset Properties, UDF, XMP e schema metadati. Il server metadati offre un’API WEB con diverse funzionalità di ricerca, come la ricerca di metadati, la similiarità delle immagini o entrambi. Per ogni risorsa restituita viene richiesto un elenco di proprietà di metadati. <!-- [More information](http://help.adobe.com/en_US/scene7/using/WS1ffbb36e209a6fc9-44279087131d3ad5622-8000.html). -->
* L’importazione dei metadati può gestire le proprietà XMP

   L’importazione dei metadati viene estesa per gestire proprietà e proprietà XMP definite negli schemi di metadati.
* Visualizzazione metadati per visualizzare solo i campi contenenti valori

   È disponibile una nuova visualizzazione metadati predefinita denominata &quot;Tutte le proprietà con valori&quot; che mostra solo le proprietà metadati contenenti valori. Le proprietà (da XMP, campi definiti dall’utente o i nuovi schemi di metadati) prive di valori sono nascoste e non verranno visualizzate.
* Le impostazioni dei metadati sono ora raggruppate nell’interfaccia utente di amministrazione.

   In **[!UICONTROL Configurazione > Impostazione applicazione]**, le pagine delle impostazioni relative ai metadati sono ora raggruppate in una sezione di livello principale denominata [!DNL Metadata]. Qui trovi le pagine di impostazione per Visualizzazione metadati, Predefiniti metadati, Schema metadati e Campi definiti dall’utente.

Questa applicazione desktop consente di importare modelli 3D esistenti e di crearli come file di vignetta 2D. I file di vignetta risultanti possono essere caricati su SPS, dove i colori e le texture possono essere sostituiti on-demand.

## Correzioni software {#section-4219d5dd0a024cd3a95aaac45598363c}

I seguenti problemi sono stati risolti in Server metadati:

* È stato risolto un problema che impediva il caricamento di file EPS di grandi dimensioni.
* Le &quot;ricerche di metadati&quot; con elenco di risultati lunghi non riescono in modo silenzioso. Questo problema causa la visualizzazione di risultati di ricerca errati.
